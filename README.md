Простое web-приложение на сервлетах, реализующее CRUD-операции над списком пользователей.

Практика с сервлетами и jsp (без бд). Для имитации модели использовалась глобальная статическая переменная, представляющая собой список пользователей.

Планируется:
1) Подключение бд и hibernate;
2) Маппинг сущности user и полное переписывание модели - с использованием dao и services.
3) Доработка сервлетов.
